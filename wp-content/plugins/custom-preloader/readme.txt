== Description ==

### Custom Preloader

Custom Preloader it's a Plugin for making your Website More Cool! This plugin runs when your Website Loads and hiding the front page until your browser download the Page Perfectly. 

This Plugin it more for the Beauty of your Website! It's hiding your front page until your objects set in their positions!
Custom Preloader has A Simple Background and a ColorFul one! You gonna love it!

### The Features

* Set Simple Background
* Set ColorFul Background
* Set Your Image
* Set Image Width - Height
* Set Margins Options
* Set Visibility
* Preview Mode
* Gradient Colors Generator installed

== Installation ==

1. Extract the `Custom-Preloader` folder to your `wp-content/plugins` directory
2. Activate the plugin through the admin interface
3. Active Simple or ColorFul Background to enable the settings 


== Frequently Asked Questions ==
= Are there any settings I can adjust? =
Yes, you have to enable Simple or ColorFul Background to activate this plugin to your Website.

= How can i change the Simple Background? =
Down to Settings Section you change the color on the Pick Simple Color.

= Can i turn off the plugin anytime i want? =
Sure! you can Disable the Simple or ColorFul Background without lose your settings!.

= why i can't enable both Settings Simple & ColorFul Background? =
only one option can be Active. ColorFul Background or Simple.

= how can i set my image? =
Just Copy your image URL and paste it to Add Your Image.

= how can i change the position of image? =
click to Positions section and play with the margins "left,top,right,bottom"
